● 创建快捷方式 在快速启动工具栏

　　在快速启动工具栏创建或删除本程序的快捷方式。

● 创建快捷方式 在 Visual Studio 工具菜单

　　在 Microsoft Visual Studio 工具菜单中创建本程序的快捷方式。
