安装卸载


● 需求

　　运行环境: Windows 95/98/ME/2000/XP/2003，Win32 X86 平台；或者 Windows XP/2003 64-Bit，X64 平台 (RawPlayer 提供纯 64 位高效代码)。

　　硬件环境: 能满足上述操作系统运行即可。为了更流畅地播放视频，推荐使用 Pentium III 或更高速的 CPU，以及高速硬盘。

　　功能限制: 如果在 Windows 95 操作系统中使用，请安装 Internet Explorer 4.0 及其以上版本。如果 CPU 不支持 MMX 指令，或者硬盘读取速度太低，将大大影响使用性能。

● 安装

　　请指定安装目录，按提示即可完成安装。

● 运行

　　执行 RawPlayer.exe 或 RawPlayer 的快捷方式，即可运行 RawPlayer。

● 卸载

　　本软件为“绿色”软件，除文件关联外，软件本身未在注册表中写入任何数据。

　　如果使用的是自解压版本，删除安装文件夹即可完成卸载操作。如果使用了创建快捷方式功能，卸载之前请从『帮助』『创建快捷方式』菜单中删除快捷方式。

　　如果使用的是安装版本，请从『控制面板』『添加删除程序』中删除本程序。
